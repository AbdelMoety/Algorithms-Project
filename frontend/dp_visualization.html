DP Formulation

1. Subproblems Definition

Let dp[i] = minimum cost to segment first i samples

Where i ∈ {0, 1, 2, ..., n} and n = signal length

2. Recurrence Relation

dp[i] = min(dp[j] + cost(segment_type, signal[j:i]))

for all j < i and segment_type ∈ {P, QRS, T}

where length constraints: L_min ≤ (i - j) ≤ L_max

3. Base Cases

dp[0] = 0 (no cost to segment empty signal)

All other dp[i] initialized to ∞

4. Solution Reconstruction

Optimal cost = dp[n]

Backtrack from dp[n] to dp[0] to reconstruct segments

Complexity Analysis

Time Complexity: O(n × m × k)

• n = signal length

• m = segment types (3: P, QRS, T)

• k = maximum segment length

Space Complexity: O(n × m)

• Storing DP table and backtracking information
